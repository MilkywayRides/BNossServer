# BNoss Server

A custom Linux distribution based on Ubuntu Server with BNoss branding.

## Features
- Prebuilt Linux kernel from kernel.org
- Minimal Ubuntu Server base
- Custom branding and configuration
- Automated builds via GitHub Actions

## Build Process
The ISO is automatically built using GitHub Actions on every push.

## Requirements
- debootstrap
- squashfs-tools
- xorriso
- isolinux
- syslinux

## Installation
Download the latest ISO from the Releases page and boot from it.
